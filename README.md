# 🪨 📄 ✂️ Stone Paper Scissors (C++)

A robust command-line implementation of the classic **Stone-Paper-Scissors** game. This project focuses on clean C++ logic, efficient input handling, and providing a smooth user experience through the terminal.

---

## 🚀 Features
- **Smart AI:** Play against a computer logic that uses randomized seed generation.
- **Input Validation:** Built-in checks to handle invalid user inputs gracefully.
- **Round-by-Round Results:** Real-time feedback on who won, lost, or tied.
- **C++ Best Practices:** Demonstrates the use of functions, loops, and conditional statements.

---

## 🛠️ Tech Stack & Requirements
- **Language:** C++
- **Compiler:** GCC (Recommended), Clang, or MSVC.
- **Environment:** Any Terminal or Command Prompt.

---

## 📂 Project Structure
- `main.cpp` : The core logic of the game.
- `README.md` : Project documentation and setup guide.

---

## 🎮 How to Play
1. **Launch** the game.
2. **Choose** your weapon:
   - `1` for **Stone**
   - `2` for **Paper**
   - `3` for **Scissors**
3. **Outcome:** The computer reveals its choice and the winner is announced immediately.
